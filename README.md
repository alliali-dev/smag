## A propos de SMAG

SMAG est une application web de gestion des établissements secondaires du système Ivoirien!
Elle est développée avec:
BACKEND
- Laravel 10
- Authentification avec laravel/breeze
- Reinitialisation de mot de passe
  
  FRONTEND
- Laravel blade
- jQuery
- axios

## Consigne d'utilisation
- Installer un serveur local(Ex: xampp, wampserver, Laravel Herd)
- Installer git bash
- Ouvrez le serveur local avec le terminal git bash
- Cloner le projet avec la commande "git clone"
- Créer une base de données MySQL,PostgreSQL...
- Créer un fichier .env à la racine du projet
- Copier le contenu du fichier .env.exemple et coller dans le fichier .env
- Ouvrez le fichier config/database.php pour voir les configurations de chaque base de données
- Utiliser les configurations de la base de données choisi pour définir ses paramètres dans le fichier .env
- Ouvrez le projet avec le terminal git bash
